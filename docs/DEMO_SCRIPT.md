### [0:15 ~ 0:50] Meeting Mode Demo — AI 임원진 토론

**화면:**
`#주간회의` 채널. 사용자(Chairman)가 메시지 입력:

```
Chairman: "이번 주 계획 세워줘"
```

**에이전트 순차 응답 (타이핑 인디케이터 표시 후 메시지 등장):**

```
COO Hudson: "이번 주 핵심 안건 3가지를 정리하겠습니다.
             1. Product Hunt 런칭 준비 — 랜딩 페이지, 티저 콘텐츠
             2. 투자자 미팅 자료 업데이트 — 최신 트랙션 반영
             3. 인프라 비용 최적화 — 현재 Azure 비용 검토"

CFO Amelia: "이번 달 예산 잔여 현황부터 공유합니다.
             운영 예산 $12,000 중 $7,200 집행 완료. 잔여 $4,800.
             Product Hunt 런칭에 추가 비용이 발생하면
             3번 안건 우선순위를 올려야 합니다."

CMO Yusef:  "Product Hunt 런칭은 화요일 오전이 최적 타이밍입니다.
             AI 카테고리 경쟁이 적은 날이에요.
             티저 콘텐츠는 X(Twitter)와 LinkedIn 동시 진행하면
             초기 트래픽을 극대화할 수 있습니다."

CFO Amelia: "Yusef, 마케팅 예산 중 유료 프로모션 배분은요?
             잔여 예산 고려하면 오가닉 중심이 안전합니다."
```

**내레이션:**

> "에이전트들이 서로 토론합니다.
> CFO는 예산을 확인하고, CMO는 마케팅 타이밍을 제안합니다.
> 그리고 CFO가 CMO의 제안에 예산 관점으로 질문합니다.
> 이건 명령-응답이 아닙니다. **진짜 회의입니다.**"

**음악:** 대화가 오갈 때 미세한 알림음 (Slack 스타일 "뚝" 소리)

> **연출 포인트:**
> - CFO Amelia가 CMO Yusef에게 **직접 질문**하는 순간을 하이라이트 (카메라 줌인 또는 하이라이트 박스)
> - 이것이 **Multi-Agent System 심사 핵심** — 에이전트 간 상호작용 증명
> - 에이전트 이름 옆에 역할 뱃지(COO/CFO/CMO)가 명확히 보여야 한다

### [0:50 ~ 1:10] Push-to-Talk + Artifact 생성

**화면:**
하단 입력 영역에 마이크 버튼이 보인다. 사용자가 **스페이스바를 홀드**한다.

**화면 액션:**
🎙️ 아이콘 활성화. 빨간 펄스 애니메이션. "Recording..." 표시.

**사용자 음성 (실제 목소리로 녹음):**

> "비용 분석 정리해줘"

**화면 액션:**
음성이 텍스트로 변환되어 채팅에 전송됨.

```
Chairman (음성): "비용 분석 정리해줘"

CFO Amelia: "정리하겠습니다. 이번 달 비용 분석표를 생성합니다."
         [📊 Excel 아티팩트 생성 중...]
```

잠시 후 Excel 아티팩트 카드가 채팅에 드롭:

```
┌──────────────────────────────────┐
│ 📊 March_Cost_Analysis.xlsx      │
│ 비용 분석표 | 3개 시트            │
│ [다운로드]  [미리보기]           │
└──────────────────────────────────┘
```

**화면 액션:** 미리보기 클릭 → Excel 내용이 테이블로 표시됨 (카테고리별 비용, 월별 추이, 잔여 예산)

**내레이션:**

> "음성으로 지시하면 CFO가 즉시 Excel을 만들어 공유합니다.
> 회의가 끝나면 산출물이 이미 준비되어 있습니다."

**음악:** 아티팩트 생성 시 "띵" 효과음 (성취감)

> **연출 포인트:**
> - 아티팩트가 "마법처럼" 생성되는 순간에 0.5초 멈추고 결과물을 보여준다
> - Excel 미리보기에 실제 숫자가 보여야 한다 — **Real-world Impact 심사 핵심**
> - 음성 → 텍스트 변환 과정이 화면에 자연스럽게 보여야 한다
