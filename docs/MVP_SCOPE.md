## 5. 기술 MVP 요구사항

### 아키텍처 개요

```
┌─────────────────────────────────────────────────────────┐
│                    Azure Static Web Apps                 │
│  ┌───────────────────────────────────────────────────┐   │
│  │  React + Tailwind CSS (Chat UI)                   │   │
│  │  ├── Meeting Room Component                       │   │
│  │  ├── Message Thread Component                     │   │
│  │  ├── Push-to-Talk Component (Web Speech API)      │   │
│  │  └── Artifact Viewer Component                    │   │
│  └───────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────┘
                         │ WebSocket (SignalR)
                         ▼
┌─────────────────────────────────────────────────────────┐
│                  Azure Functions (Node.js)               │
│  ├── HTTP Trigger: /api/chat                            │
│  ├── HTTP Trigger: /api/meeting                         │
│  ├── HTTP Trigger: /api/artifact                        │
│  └── SignalR Binding: Real-time broadcast               │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│              Semantic Kernel (Agent Layer)               │
│  ├── COO Agent (Hudson) ──┐                               │
│  ├── CFO Agent (Amelia) ──┤── A2A Message Passing         │
│  └── CMO Agent (Yusef) ──┘                               │
│                                                         │
│  Plugins:                                               │
│  ├── MeetingMinutesPlugin (Markdown generation)         │
│  └── ExcelArtifactPlugin (.xlsx generation)             │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│           Azure AI Foundry (GPT-4o)                     │
│           Model endpoint for all agents                 │
└─────────────────────────────────────────────────────────┘
```
